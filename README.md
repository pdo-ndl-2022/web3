Sido Web3

Sido Web3 is a library for interacting with Ethereum smart contracts from the browser. It is a wrapper around the Web3.js library.

## Production

How to use it:

```bash
$ docker build . -t sidoweb3
```

```bash
$ docker run -p 80:80 sidoweb3:latest
```

## Development

How to use it:

```bash
$ npm install
```

```bash
$ npm run dev
```
